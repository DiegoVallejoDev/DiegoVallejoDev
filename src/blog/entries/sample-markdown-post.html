El Formato Markdown

Este post está escrito en Markdown, lo que demuestra la capacidad del blog para procesar y convertir archivos .md a HTML. Markdown es perfecto para escribir contenido porque:

  1. Es simple: La sintaxis es limpia y fácil de leer
  2. Es portable: Los archivos se pueden usar en múltiples plataformas
  3. Es enfocado: Te permite concentrarte en el contenido, no en el formato

Tecnología del Blog

Este blog está construido con:

  • Parcel como bundler para un desarrollo rápido
  • Marked para convertir Markdown a HTML
  • Diseño responsive con tema claro/oscuro
  • Sin frameworks pesados, solo HTML, CSS y JavaScript vanilla
